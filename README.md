# 🌸 AETHER AURA – Luxury Perfume Atelier

AETHER AURA is a premium luxury perfume brand website that offers an immersive digital experience for fragrance enthusiasts. Inspired by high-end perfume ateliers, the platform allows users to explore exclusive fragrance collections, design bespoke perfumes, take a personalized scent quiz, and book private consultations with master perfumers.

## ✨ Features

* Elegant luxury-inspired user interface
* Interactive perfume customization (Scent Lab)
* Personalized fragrance recommendation quiz
* Curated luxury perfume collections
* Custom bottle engraving and design
* Concierge consultation booking
* Shopping cart and checkout simulation
* Fully responsive design
* Smooth animations and interactive effects

## 🛠️ Tech Stack

* HTML5
* Tailwind CSS
* JavaScript (ES6)
* Google Fonts
* Material Icons

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/aether-aura.git
```

2. Open the project folder

```bash
cd aether-aura
```

3. Open `index.html` in your browser or run it using Live Server in Visual Studio Code.
